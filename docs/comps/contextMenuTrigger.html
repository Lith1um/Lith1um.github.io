Context Menu Triggers

Canopy context menu triggers are used to wrap content that should show a custom menu when the context menu event is fired. They are referenced using the<cpy-context-menu-trigger>custom HTML element. This component takes an array of items, and supports item clicks either navigating to links/paths or running custom functions if you need side-effects or a different behaviour to a traditional link.
