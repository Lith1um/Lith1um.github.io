Navigation Menus

Canopy navigation menus show a list of navigation items. They are referenced using the custom HTML element. This component takes an array of items, and supports item clicks either navigating to links/paths or running custom functions if you need side-effects or a different behaviour to a traditional link.

Custom Styles

Through the use of CSS variables, you can customise Canopy components to achieve the look and feel you want. There are currently no styling variables for this component. You can learn more about Canopy colors and root configuration here.
