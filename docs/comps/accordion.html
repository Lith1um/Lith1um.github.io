Accordions

Canopy accordions provide an expandable details-summary view and are referenced using the custom HTML element. The component utilizes the slot element, allowing you to render your own HTML in the accordion content.

Accessibility

Canopy Accordions don't come with any accessibility treatment, because it doesn't have control over its markup.
