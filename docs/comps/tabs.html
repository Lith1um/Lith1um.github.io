Tabs

Canopy Tabs organize content into separate views where only one view can be visible at a time. They are referenced using the<cpy-tabs>,<cpy-tab-header>and<cpy-tab-content>custom HTML elements. Each tab's label is shown in the tab header and the active tab's label is designated with the animated ink bar. When the list of tab labels exceeds the width of the header, pagination controls appear to let the user scroll left and right across the labels.
