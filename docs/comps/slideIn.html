Slide Ins

Canopy slide ins provide an animated entry to elements when scrolled into view. They are referenced using the custom HTML element. The component utilizes the slot element, allowing you to render your own items in the slide in.

Custom Styles

Through the use of CSS variables, you can customise Canopy components to achieve the look and feel you want. There are currently no styling variables for this component. You can learn more about Canopy colors and root configuration here.

Host styles

Canopy components are designed to have their host element styled directly so that it can be overridden for specific implementation purposes.This is generally used to adjust , , , , , , etc.

There are likely other properties that you can alter on the host element for each individual component, open your dev tools and inspect each component to see what you can override on the host!

Accessibility
