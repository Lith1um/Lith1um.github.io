Context Menus

Canopy context menus show a list of options. Their purpose is generally to populate a with a list of items when the container is right-clicked. They are referenced using the custom HTML element. This component takes an array of items, and supports item clicks either navigating to links/paths or running custom functions if you need side-effects or a different behaviour to a traditional link.
