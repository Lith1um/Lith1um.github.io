Cards

Canopy cards provide a basic wrapper for content and are referenced using the <cpy-card> custom HTML element. The component utilizes the slot element, allowing you to render your own HTML in the card.

Host styles

Canopy components are designed to have their host element styled directly so that it can be overridden for specific implementation purposes.This is generally used to adjust display, position, background-color, padding, margin, width, height etc.

There are likely other properties that you can alter on the host element for each individual component, open your dev tools and inspect each component to see what you can override on the host!

Accessibility

Cards serve a wide variety of scenarios and may contain many different types of content. Due to this flexible nature, the appropriate accessibility treatment depends on how you use <cpy-card>.
