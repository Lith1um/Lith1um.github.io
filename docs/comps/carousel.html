Carousels

Canopy carousels provide a scrolling list of items that can be scrolled manually or by using buttons in the top right and are referenced using the <cpy-carousel> custom HTML element. The component utilizes the slot element, allowing you to render your own items in the carousel.

Custom Styles

Through the use of CSS variables, you can customise Canopy components to achieve the look and feel you want. There are currently no styling variables for this component. You can learn more about Canopy colours and root configuration here.

Accessibility

Carousels serve a wide variety of scenarios and may contain many different types of content. Due to this flexible nature, the appropriate accessibility treatment depends on how you use <cpy-carousel>.
