Stacks

Canopy stacks display a stacked list of elements that when hovered over will expand to no longer overlap. They were originally designed to should a list of avatars (as seen below). They are referenced using the <cpy-stack> custom HTML element. This component utilises the slot element, allowing you to stack any elements you want. The current behaviour will mean that each item in the stack is shifted by half its width.
