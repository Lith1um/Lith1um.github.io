Avatars

Canopy avatars provide a simple way to represent the user, and displays the profile picture in a consistent way. They are referenced using the <cpy-avatar> custom HTML element.

Custom Styles

Through the use of CSS variables, you can customise Canopy components to achieve the look and feel you want. Below is a list of variables you can use to configure this component. You can learn more about Canopy colours and root configuration here.

Accessibility

Avatars serve a wide variety of scenarios and may contain many different types of content. Due to this flexible nature, the appropriate accessibility treatment depends on how you use <cpy-avatar>.
