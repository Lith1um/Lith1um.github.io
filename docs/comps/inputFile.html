File Inputs

Canopy file input allow you to select files for upload through a form element. They are referenced using the custom HTML element.

The Canopy file input accept property allows you to restrict what files you allow users to upload and mimics the HTML file input element. Learn more about restricting file types with file inputs.

Canopy form inputs use a custom validation factory to provide built-in validator functions for your inputs. They can also be extended upon to create your own validators.

Learn more about form validation in Canopy.
