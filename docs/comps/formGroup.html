Form Groups

Canopy form groups provide a basic wrapper around canopy form inputs. They extend the base methods of inputs, allowing you to run validation and trigger/reset errors on the whole form with ease. They are referenced using the custom HTML element.

Additionally, a basic flex layout is used to display the inputs and can be easily overridden using the host styles. Inputs to the form group also allow you to change the size and disabled state of all canopy inputs within the form group.
