Tables

Canopy tables are used to organize and display data efficiently. They are referenced using the<cpy-table>custom HTML element. The table will by default pick the object keys of the first element to create the headers.

Custom Styles

Through the use of CSS variables, you can customise Canopy components to achieve the look and feel you want. Below is a list of variables you can use to configure this component. You can learn more about Canopy colours and root configuration here.

Accessibility

Tables serve a wide variety of scenarios and may contain many different types of content. Due to this flexible nature, the appropriate accessibility treatment depends on how you use <cpy-table>.
