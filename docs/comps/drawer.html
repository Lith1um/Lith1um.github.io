Drawer Containers

Canopy drawers provide a toggle-able drawer to a container. They are referenced using the<cpy-drawer-container>custom HTML element. This component utilises the slot element, allowing you to render your own HTML in the drawer. On Mobile screens, the drawer will open to the full width of the screen, overlaying the main content.

Custom Styles

Through the use of CSS variables, you can customise Canopy components to achieve the look and feel you want. There are currently no styling variables for this component. You can learn more about Canopy colours and root configuration here.
