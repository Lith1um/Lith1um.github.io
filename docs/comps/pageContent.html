Page Contents

Canopy page content components are stylised components for blog/doc style content. They are referenced using the<cpy-page-content>custom HTML element. This component utilises the slot element, allowing you to render your own HTML in the content. Any direct child headers found with an id will have an entry added to the contents list. On mobile the contents list is hidden to preserve screen real-estate.

Page Contents serve a wide variety of scenarios and may contain many different types of content. Due to this flexible nature, the appropriate accessibility treatment depends on how you use <cpy-page-content>.
