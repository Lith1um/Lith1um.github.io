Toasts

Canopy toasts provide well defined elements for rendering messages that appear from off screen. They are referenced using the <cpy-toast> custom HTML element. This component utilizes the slot element, allowing you to render your own HTML in the toast, however it is best used in conjunction with the <cpy-alert> component.
