Angular Forms

Canopy provides the necessary implementations to work with angular forms in a way developers will find familiar. Both template forms and reactive forms are supported, but if you'd prefer to bind to the built in inputs and outputs on the form component manually then that is equally possible. Let's dive into some examples.
